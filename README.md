# Advent of Code
Look up the challenges here - https://adventofcode.com/

# Run

Each day are in a separate folder, the `index.ts` file the start of the solution, the `data` folder is for the inputs, and the `models` are for the data models.

To run the solution type `npm start -- ./2022/05/index.ts`, input the right folder for the right challenge.
# Other
